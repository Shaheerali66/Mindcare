# MindCare 💗

**Your Safe Space to Talk, Heal & Grow**

MindCare is a private online mental health counseling platform designed for students and young people. It connects users with affordable psychology graduate counselors and includes an AI chatbot for basic guidance.

## Features

- 🧑‍🎓 **Student Dashboard** — Browse counselors, book sessions, AI chatbot, mood tracking
- 👨‍⚕️ **Consultant Dashboard** — Manage schedule, student notes, earnings, profile
- 👑 **Admin Dashboard** — User management, counselor approvals, support queries, settings
- 🤖 **AI Chatbot** — 24/7 mental health guidance with keyword-based responses
- 🔒 **Privacy First** — End-to-end encrypted, judgment-free platform
- 💰 **Affordable Plans** — Starting from Rs. 500/month

## Tech Stack

- React + Vite
- React Router DOM
- Framer Motion (Animations)
- React Icons
- Vanilla CSS (Premium Design System)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
